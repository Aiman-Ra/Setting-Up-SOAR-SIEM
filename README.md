<h1>SIEM with Wazuh, TheHive and SOAR through Shuffle</h1>


<h2>Description</h2>
This project consists of setting up a SIEM through cloud services and VMs. Wazuh and TheHive will assist us in case managment.
<br />


<h2>Utilities Used</h2>

- <b>Wazuh</b>
- <b>TheHive</b>
- <b>Shuffle</b>
- <b>Virtualbox</b>

<h2>Environments Used</h2>

- <b>Azure/Digital Ocean</b>
- <b>Windows 10</b> (21H2)
- <b>Ubuntu 22.04 LTS</b> (21H2)
